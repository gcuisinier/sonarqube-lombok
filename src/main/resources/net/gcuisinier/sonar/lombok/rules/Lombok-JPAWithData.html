This rule detects usage of Lombok @Data annotation in JPA @Entities

You should avoid the meta @Data annotation and prefer to use @Getter, @Setter, @ToString annotations instead

Noncompliant Code Example

@Entity
@Data
public  class Entity {

    @Id
    @GeneratedValue(strategy= GenerationType.AUTO)
    private Long id;

    private String firstName;

    private String lastName;

    @OneToMany
    private Set relation;

}

Compliant Solution

@Entity
@Getter
@Setter
@ToString
@RequiredArgsConstructor
@NoArgsConstructor
public class Entity {

    @Id
    @GeneratedValue(strategy= GenerationType.AUTO)
    private Long id;

    private String firstName;

    private String lastName;

    @OneToMany
    @ToString.Exclude
    private Set relation;

}
