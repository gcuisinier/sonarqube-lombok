This rule detects usage of Lombok @Builder annotation in classes that extend another class.

By default, @Builder doesn't include fields from the parent class, which can lead to incomplete objects being created. Use @SuperBuilder instead when working with inheritance.

Noncompliant Code Example

class ParentClass {
    private String parentField;
}

@Builder // Noncompliant
class ChildClass extends ParentClass {
    private String childField;
}

Compliant Solution

class ParentClass {
    private String parentField;
}

@SuperBuilder
class ChildClass extends ParentClass {
    private String childField;
}
